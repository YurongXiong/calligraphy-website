# 项目记忆 - 书法家网站 calligraphy-app

## 项目背景

这是一个中文书法作品创作网站，用户可以：
1. 选择作品形式（春联/挂画/牌匾）
2. 输入文字内容
3. 在手机上逐字手写
4. 选择笔触风格和模板
5. 预览并导出作品图

## 当前进度（截至 2026-03-21）

### 已完成功能
- 全部 6 个页面：首页、输入页、书写页、风格页、预览页、导出页
- 手机浏览器触摸书写（perfect-freehand 笔触引擎）
- 草稿本地保存/恢复
- 三种笔触风格（自然/浓墨/飞白）
- 六种模板（每种作品形式 2 个）
- PNG/JPG 导出

## 技术要点

- Next.js 16 + React 19 + TypeScript + Tailwind CSS 4
- Zustand store 持久化到 localStorage
- DPR 缩放处理高清屏
- 触摸事件直接绑定原生事件，不用 React onPointer*
